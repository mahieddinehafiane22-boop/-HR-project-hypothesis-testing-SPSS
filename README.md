# -HR-project-hypothesis-testing-SPSS
A project to test if the quality of life  impacts the overall well being of employees at NAFTAL using real world data gathered by distributing surveys to a sample of 40 employees from a population of 131 personnels. 
THIS PROJECT WAS CONDUCTED AS A PART OF AN INTERNSHIP PROGRAM.
The full analysis/interpretations can be found on the PDF file along with the SPSS inputs and outputs.
